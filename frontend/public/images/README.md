# Banner Images

Please add the following images to this directory:

1. **banner-visa.jpg** - Visa stamp image (blue/white/red passport visa)
2. **banner-car.jpg** - Car navigation/driving image (night driving with GPS)
3. **banner-business.jpg** - Business/cityscape image (modern city skyline with waterway)
4. **banner-grocery.jpg** - Ethiopian food image (injera, misir wot, gomen on mesob)

## Image Requirements:
- **Format**: JPG or PNG
- **Recommended size**: 800x400px or larger (will be cropped to fit)
- **Aspect ratio**: 2:1 (width:height) works best

## Usage:
The images will be used as backgrounds for the community banner items on the landing page. The text will be overlaid on top with proper contrast using brand colors (deep royal blue overlay and gold accents).

If images are not provided, the banners will display with a solid deep royal blue background as a fallback.
